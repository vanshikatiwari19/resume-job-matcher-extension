# Resume Job Matcher Chrome Extension

## 🚀 Features
- Upload resume (PDF)
- Extract skills
- Job role matching
- Skill gap analysis

## 🛠️ Tech Stack
- JavaScript
- HTML, CSS
- Chrome Extension API

## 💡 Description
This Chrome extension analyzes a user's resume and suggests relevant job roles based on skills. It also highlights missing skills required for each job.

## ▶️ How to Use
1. Upload your resume
2. Click Analyze
3. View job matches and skill gaps
